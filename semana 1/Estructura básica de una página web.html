<!DOCTYPE html>

Define que el documento esta bajo el estandar de HTML 5

Parametros:

<!DOCTYPE [Elemento de primer nivel] [Disponibilidad] "[+/-]//[Organizacion]//[Tipo] [Nombre]//[Idioma]" "[URL]">

DOCTYPE: Indicamos que estamos ante la declaración del DOCTYPE Elemento de primer nivel: Tenemos que escribir el elemento de nivel superior declarado en el DTD. En nuestro caso es html Disponibilidad: Indicamos la disponibilidad del identificador. En nuestro caso tenemos PUBLIC porque se utilizará un identificador público, aunque también existe SYSTEM (cuando se utiliza un identificador de sistema) o URL (cuando el identificador es local o URL). +/-: Según la organización se utilizará + o -, en el caso de la W3C se utiliza – Organización: La organización que ha creado el DTD que vamos a utilizar. Tipo: El tipo de documento que estamos incluyendo, o sea, DTD. Nombre: El nombre del DTD. En nuestro ejemplo HTML 4.01 Idioma: El idioma en el que está escrito el DTD, normalmente es EN (inglés) URL: La URL del DTD.

<html></html>

Representa la raíz de un documento HTML o XHTML. Todos los demás elementos deben ser descendientes de este elemento.

Parametros: incluyen los atributos globales

<head></head>

Define la cabecera de una página o sección. Usualmente contiene un logotipo, el título del sitio Web y una tabla de navegación de contenidos.

Parametros: incluyen los atributos globales

<title></title>

Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada

Parametros: incluyen los atributos globales

<body></body>

Representa el contenido principal de un documento HTML. Solo hay un elemento en un documento.

Parametros: incluyen los atributos globales y atributos como TEXT, BGCOLOR, BACKGROUND, LINK, VLINK, ALINK

<h1></h1>

H1 es el elemento HTML utilizado, de manera habitual, para identificar la cabecera más importante en una página web. Los elementos de encabezado implementan seis niveles de encabezado del documento, h1 es el más importante, y h6, el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta.

Parametros: incluyen los atributos globales y atributo align

<p></p>

Define una parte que debe mostrarse como un párrafo

Parametros: incluyen los atributos globales y atributo align

Atributos globales

Los atributos globales son atributos comunes a todos los elementos HTML; se pueden usar en todos los elementos.

Atributos: ACCESSKEY, AUTOCAPITALIZE, AUTOCOMPLETE, CLASS, CONTENTEDITABLE, CONTEXTMENU, DIR, DRAGGABLE, ENTERKEYHINT, HIDDEN, INPUTMODE, ID, IS, ITEMID, ITEMPROP, ITEMREF, ITEMSCOPE, ITEMTYPE, LANG, NONCE, SLOT, SPELLCHECK, STYLE, TABINDEX, TITLE, TRANSLATE, DROPZONE.
